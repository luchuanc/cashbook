- 已按用户补充要求调整新版登录/注册页：注册按钮下方不再展示返回按钮或其他内容，返回登录入口移到页面顶部。
- 已根据独立架构复查修复阻塞项：
  - `/m/*` 未登录直达时，无论 SSR/CSR 均跳转 `/m/login`。
  - 新版首页无账本时不再请求 `daily()`、预算和今日流水接口。
  - 快速记账无账本时不再尝试写入流水，并提示先选择账本。
- 已为新版流水、分析、预算、待收款、记账配置等页面补充无账本边界防护。
- 已将 `doApi` 通用 400 登录跳转按当前 UI 模式路由到 `/login` 或 `/m/login`，避免移动端会话失效时绕回旧登录页。
- 已多次运行 `npm run build`，构建通过。
